# Image Downloader

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/image-downloader.git
```

### Step 2

Change the directory to image-downloader

```shell
cd image-downloader
```

### Step 3

Install requirements

```shell
pip install -r requirements.txt
```

### Step 4

Run project

```shell
python main.py
```
